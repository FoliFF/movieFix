Movie API Details

This will return details of movies to the user.
This will allow users to query specific movie types, genres, and actors.
You will be able to sign up, update your personal information and make a list of your favorite movies.

POST /users Allows user to register
PUT /users/[id] Allows user to update their information
POST /users/[id]/[title] Allows user to add movie to their list of favorites
DELETE /users/[name] Allows user to delete the account
